# Commandes {{site.data.keyword.registrylong_notm}} (`ibmcloud cr`) pour la gestion des images Docker dans votre espace de nom
{: #registry_cli_reference}

Vous pouvez utiliser le plug-in d'interface de ligne de commande `container-registry` pour configurer votre propre espace de nom d'images dans un registre privé, hébergé et géré par IBM, dans lequel vous pouvez stocker et partager de manière sécurisée des images Docker avec tous les utilisateurs de votre compte {{site.data.keyword.cloud}}.
{:shortdesc}

## Commandes `ibmcloud cr`
{: #registry_cli_reference_bxcr}

Exécutez les commandes `ibmcloud cr` dans l'interface de ligne de commande d'{{site.data.keyword.registryshort_notm}}.
{:shortdesc}

Voir [Interface CLI {{site.data.keyword.registrylong_notm}}](/docs/services/Registry?topic=container-registry-cli-plugin-containerregcli#containerregcli).
